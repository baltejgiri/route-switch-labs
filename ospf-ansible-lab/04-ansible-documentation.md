# Ansible Documentation

Ansible has great [documentation](https://docs.ansible.com/projects/ansible/latest/index.html) on their website. Ansible is used by many vendors for networks, systems and other automation tasks. Ansible documentation has modules that can be viewed on web or CLI, using command ```ansible-doc``` on a linux operating system.

Output shows all the modules available for keyword *cisco.ios*, each line in the output is a ansible module. Based of the module you're working with it can be further seeing in details.

Following command can be used to view the output of the module we're working with.

```bash
$ ansible-doc cisco.ios.ios_config
```
This command displays the details of the module *cisco.ios.ios_config*. Module can be read to find how to use the specific details in your own playbook which we will run in upcoming parts of this lab.
